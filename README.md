# Escape-from-Noro-th-Quad
Computer game for GameMaker Player where you try to escape Noro-infected students... including your friend!

System requirements:
PC, GameMaker (Lite or later versions), GameMaker Player

Game description:
You are a student at the University of Michigan.
There has been an outbreak of the Noro virus!
Fellow students will cough and sneeze, spreading disease.
Use the hand sanitizer to stop the virus from spreading.
Friends are nice, but you can't let them infect you!
